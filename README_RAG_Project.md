# RAG Application using LangChain, OpenAI API, and FAISS

This repository contains a beginner-friendly Retrieval-Augmented Generation (RAG) project built in Jupyter Notebook.

## Project Overview

The notebook demonstrates a complete RAG pipeline:

1. Load a text document
2. Split the text into chunks
3. Convert chunks into embeddings
4. Store embeddings in FAISS
5. Retrieve relevant chunks for a user question
6. Generate an answer using an OpenAI chat model

## Tech Stack

- Python
- Jupyter Notebook
- LangChain
- OpenAI API
- FAISS

## File

- `RAG_Application_using_Langchain_OpenAI_API_and_FAISS_GitHub_Ready.ipynb`

## How to Run

1. Install the required packages:
   ```bash
   pip install -r requirements_rag_project.txt
   ```
2. Set your `OPENAI_API_KEY`
3. Open the notebook in Jupyter or Google Colab
4. Run the cells step by step
